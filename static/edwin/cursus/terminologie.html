Terminologie

Copyright © 2001 Edwin Martin <edwin@bitstorm.nl>

Laatst aangepast: 12 februari 2002.

Deze lijst met terminologie hoort bij de volgende cursussen:

  • Cursus Dynamic HTML
  • Cursus Cascading Style Sheets (CSS)

De lijst

ADSLAsymmetric Digital Subscriber Line. Een techniek voor snel netwerkverkeer over een eenvoudige koperen telefoonlijn.
Alpha kanaalEen kanaal, net als de kleurenkanalen, dat de doorzichtigheid van een afbeelding bepaalt.
Anti-aliasHet overvloeien van de randen van een figuur naar de achtergrond waardoor gekartelde randen worden voorkomen.
ApacheMeest gebruikte webserver. Zie de Apache homepage.
ASCIIAmerican Standard Code for Information Interchange. Lettertekenset van 96 lettertekens en 32 regelcode's, wordt ondersteund door alle moderne computers.
ASPActive Server Pages. Microsoft's server side scripting taal voor webpagina's. Zie ook de MSDN ASP pagina.
AttribuutEen parameter van een HTML-tag. Zie voor een uitleg de CSS-cursus.
BinairGetal in het tweecijferig stelsel. Elk cijfer is nul of één, bijvoorbeeld 101100010110.
BitBinary digit. Kleinste hoeveelheid informatie en kan maar twee waarden bevatten, bijvoorbeeld waar en onwaar.
BroncodeDe HTML- en CSS-code van een webpagina of de programmacode van een programma. Gewoonlijk geschreven door de auteurs, maar het kan ook door een programma gegenereerd zijn.
ByteBy Eight. Groep van acht bits.
CVeel bebruikte low-level prorammeertaal.
C++Object-georiënteerde uitbreiding van C.
CommunicatorNaam van Netcape's browser Navigator inclusief de mail- nieuws- en opmaakonderdelen. Sinds versie 6 heet de browser ook Netscape.
CookieEen kleine hoeveelheid informatie dat een website kan opslaan in de browser en later weer kan uitlezen. Meer informatie op Cookie Central.
CSSCascading Style Sheets, ofwel Overlappende stijlbladen. Met CSS kan van elke HTML-tag nauwkeurig de vorm worden beschreven. Zie ook de CSS-cursus en de W3C CSS homepage.
DAVZie WebDAV.
DecimaalGetal in het gebruikelijke tiencijferig stelsel. Elk cijfer is er een van nul tot en met negen.
DHTMLDynamische HTML. Met de combinatie van HTML, CSS, JavaScript en DOM is het mogelijk om interactieve, dynamische webpagina's te ontwikkelen. Zie ook de DHTML-cursus.
DOMDocument Object Model. Beschrijft de struktuur van een document, zodat het met een programmeertaal te benaderen is. Zie ook de W3C DOM homepage.
DownloadenHet direct online ophalen van software van een andere computer naar de eigen computer.
DTDDocument Type Definition. Beschrijft de elementen (tags) en attributen en hoe ze gebruikt mogen worden. Een voorbeeld is de HTML 4.01 DTD.
DTPDesktop publishing. Het opmaken en vormgeven van publicaties vanaf de bureaucomputer.
ECMAScriptDe onafhankelijke standaard voor JavaScript. Zie ook ECMAScript Language Specification.
ElementHet type HTML-tag. Zie voor een uitleg de CSS-cursus.
EntityEen tekenreeks in SGML (en dus ook HTML en XML) die vervangen kan worden. In HTML wordt bijvoorbeeld &euml; vervangen door ë.
FlashEen browser-plugin en technologie van Macromedia waarmee zeer mooie, interactieve animaties ontwikkeld en getoond kunnen worden. Zie de Flash homepage.
FontEen lettertype. Voorbeelden van lettertypes zijn: Helvetica, Times New Roman en Comic Sans.
GIFGraphics Interchange Format. Beeldopslagformaat voor afbeeldingen met maximaal 256 kleuren.
HexadecimaalGetal in het zestiencijferig stelsel. Elk cijfer is er een van nul tot en met negen of A tot en met F, Bijvoorbeeld 2E6AF0.
HTMLHyperText Markup Language. De opmaaktaal voor het web. Zie ook de W3C HTML homepage.
HTTPHypertext Transfer Protocol. Gegevensoverdrachtprotocol voor het WWW.
HyperlinkWoorden en afbeeldingen kunnen met een hyperlink aan een andere webpagina gekoppeld worden. Door op het woord of de afbeelding te klikken, gaat de browser naar de andere pagina. In plaats van het gaan naar een andere webpagina, kan ook naar bijvoorbeeld een programma worden gelinkt, die dan gedownload wordt. Zie ook URL.
IEZie Internet Explorer.
IISInternet Information Services. Microsoft's webserver. Zie ook de Windows 2000 server documentation.
InternetGeheel van netwerken die met elkaar zijn verbonden via het TCP/IP-protocol.
Internet ExplorerDe webbrowser van Microsoft. Zie ook de IE homepage.
JavaEen door Sun ontwikkelde programmeertaal. Zie ook de Java homepage.
JavaScriptEen door Netscape ontwikkelde scripting-taal, waarmee webpagina's geprogrammeerd kunnen worden.
JPEGJoint Photographic Experts Group. Beeldopslagformaat dat erg geschikt is voor foto's. Zie ook de JPEG homepage.
JScriptMicrosoft's JavaScript-compatible scriptingtaal. Zie Microsoft Scripting Technologies.
KabelMet kabel wordt vaak Internet via de TV-kabel bedoeld.
LynxEen browser met een tekst-interface. Zie ook Lynx information.
MozillaDe open-source variant van Netscape 6. Zie ook de Mozilla homepage.
MSIEZie Internet Explorer.
NavigatorOfficiële naam van de Netscape browser. Sinds versie 6 heet de browser ook Netscape.
NetscapeBedrijf dat in 1994 de eerste populaire webbrowser ontwikkelde en tevens de naam van hun laatste browser. Zie ook de Netscape homepage.
OnlineVia een actieve netwerkverbinding. Bijvoorbeeld via een telefoonlijn, kabel of ADSL.
Open SourceSoftware met vrijgegeven broncode. Zo kunnen anderen de software aanpassen en verbeteren. Zie ook de Vereniging Open Source Nederland.
OperaDe snelle no-nonsense browser. Zie de Opera homepage.
PDFPortable Document Format. Een door Adobe ontwikkeld documentformaat. Het kan gelezen en afgedrukt worden met het gratis Acrobat Reader. Zie ook Adobe PDF.
PerlScripting taal. Zie ook de Perl Mongers homepage.
PHPServer side scripting taal voor webpagina's. Zie ook de PHP homepage.
PixelPicture Element. Pixels zijn de 'puntjes' waaruit een digitale afbeelding is opgebouwd.
PNG Portable Network Graphics, spreek uit: ping. Beeldopslagformaat vergelijkbaar met het GIF-formaat, maar zonder de licentie- en kleurbeperkingen. Zie ook de PNG homepage en Unisys/CompuServe GIF Controversy.
PythonScripting taal. Zie ook de Python homepage.
QuicktimeMediaformaat, ontwikkeld door Apple. Zie de Quicktime homepage.
RDFResource Description Framework. XML-taal dat metagegevens (gegevens over gegevens) beschrijft. Zie ook de W3C RDF homepage.
RealRealVideo en RealAudio zijn streaming media formaten die met de RealPlayer bekeken kunnen worden. Zie ook de Real homepage.
RGBRood-Groen-Blauw. Alle kleuren op een beeldscherm zijn uit deze kleuren opgebouwd.
ScriptingtaalProgrammeertaal waarvan de broncode het programma is. Voorbeelden zijn JavaScript, VBScript en Perl.
SGMLStandard Generalized Markup Language. SGML is een ISO-standaard opmaaktaal, waarvan HTML en XML zijn afgeleid.
SMILSynchronized Multimedia Integration Language. XML-implementatie dat interactieve, gesynchroniseerde multimedia beschrijft en geschikt is voor het Internet. Zie ook de CWI SMIL Page en de W3C specificatie.
Streaming mediaGeluid en Video dat wordt afgespeeld terwijl de data binnenkomt. Het is dus niet nodig eerst een bestand te downloaden.
SVGScalable Vector Graphics. XML-implementatie dat vectorgrafieken beschrijft. Zie ook de SVG specificatie.
UnicodeUnicode is een lettertekencodering voor elk bekend schrift. Een Unicode-tekst kan dus bijvoorbeeld Koreaanse en Arabische teksten bevatten. Zie ook de Unicode homepage.
UploadenHet direct online versturen van software van de eigen naar een andere computer.
URLUniform Resource Locator. Beschrijft de plaats en de wijze van toegang tot een bron op het Internet. Meestal is dat een webpagina, maar het kan bijvoorbeeld ook een programma of geluidsbestand zijn. Zie ook Hyperlink, RFC 1738 en W3C Naming and Addressing Overview.
VBScriptVan Visual Basic afgeleide scriptingtaal van Microsoft. Wordt zelden op webpagina's toegepast. Zie Microsoft Scripting Technologies.
W3CWorld Wide Web Consortium. Deze organisatie ontwikkelt standaarden voor het web. Zie ook de homepage.
WAPWireless Application Protocol. Gegevensoverdrachtprotocol en applicatieomgeving voor mobiele apparaten. Zie ook de Wap Forum homepage.
WebDAVWeb-based Distributed Authoring and Versioning. Uitbreiding van het HTTP-protocol waarmee door meerdere personen websites kunnen worden beheerd. Zie ook WebDAV Resources
WMLWireless Markup Language. XML-opmaaktaal, ontwikkeld voor mobiele apparatien. Zie ook de Wap Forum homepage.
WWWWorld Wide Web. De door hyperlinks aan elkaar gekoppelde webpagina's vormen samen het WWW.
XHTMLHTML, uitwisselbaar met XML. Zie ook de W3C XHTML Recommendation.
XMLExtensible Markup Language. Universeel uitwisselingsformaat voor gestructureerde documenten en gegevens. Zie ook de W3C XML Recommendation.
ZIPFormaat om bestanden in te pakken en comprimeren (verkleinen). Voor het in- en uitpakken bestaan diverse shareware en freeware programma's. Bijvoorbeeld het freeware PowerArchiver 2000.
