Java PC-emulator

ma 18 aug 2008 22:36

Eens in de zoveel tijd kom je iets tegen dat de grenzen van de techniek weer een stukje opschuift. Een aantal jaren geleden waren dat Virtual Machines, PC-emulatoren en andere dingen die nu gemeengoed zijn.

De volgende stap is alweer gezet: het draaien van een PC-emulator in een Java Virtual Machine. Je kunt bijvoorbeeld Linux draaien in een applet, in je browser. Of, wat ik zelf heb geprobeerd, DOS draaien in de Java VM op mijn mobiele telefoon. Het werkt: je draait een heuse DOS op je telefoon waarin je DOS-commando's kunt geven.

Het project is nog sterk in ontwikkeling en van de grafische kracht moet je nog niet teveel verwachten (een spelletje Mario is mogelijk). Als dit project zich nog wat jaren doorontwikkeld, kan het nog heel interessant worden. Een programma uitproberen, niet door het te installeren, maar in een applet in je browser. Of Windows of MacOSX draaien op een niet-Intel processor. Mogelijkheden genoeg. Nu maar hopen dat het project niet doodbloed, zoals bij dit soort projecten wel eens gebeurd.
