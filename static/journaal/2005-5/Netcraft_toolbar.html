Netcraft toolbar

wo 25 mei 2005 22:02

Vandaag kwam ik achter het bestaan van de Netcraft toolbar. Deze toolbar (gereedschapsbalk) kun je in je browser opnemen, waarna informatie wordt getoont over de website die je op dat moment bekijkt.

Het belangrijkste doel van de toolbar is om te dienen als een graadmeter van betrouwbaarheid. Dus als je een beetje wantrouwend bent op Internet (en veel mensen zouden dat moeten zijn), dan is deze toolbar een mooie uitkomst.

Ik vind de toolbar vooral leuk omdat ik zie waar ik ben: zowel het land als de hoster worden gemeld.

De toolbar kan gratis worden gedownload in werkt in zowel Internet Explorer als Firefox.
