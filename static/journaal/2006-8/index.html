Bryce

di 8 aug 2006 23:13

Zo'n tien jaar geleden zag ik Bryce voor het eerst: software waarmee je in heel korte tijd heel indrukwekkende plaatjes kan maken. Klik: je hebt een zee, klik: een berg en klik: een mooie wolkenlucht. Even wat kleuren en andere parameters aanpassen en presto: in tien minuten een kunstwerkje.

Zodra ik van Jolie de lage prijs hoorde ($ 99,-) en op de website ontdekte dat je nu ook mensenfiguren in de tekening kan opnemen en animaties kan maken, heb ik het gelijk gekocht.

Hierboven zie je twee tekeningen die ik afgelopen weekend heb gemaakt (tijd: een paar uurtjes).

Oh, ja: de tijd om een plaatje van een hoge kwaliteit te laten uitrekenen kost de computer tegenwoordig geen tien uur meer, maar een kwartiertje.

  • Bryce 5.5
