Firefox 1.0 uitgebracht

do 11 nov 2004 22:28

Twee dagen geleden is eindelijk de webbrowser Firefox 1.0 uitgebracht.

Er is een goede kans dat je denkt wat er nu zo goed is aan Firefox en waarom je het zou moeten gebuiken.

Firefox komt voort uit de Mozilla browser, die op zijn beurt een opnieuw ontwikkelde Netscape 4 is. Firefox is echter ontdaan van alles wat niet in een browser hoort zoals het e-mail programma. De gebruiksvriendelijkheid is daarentegen sterk verbeterd. Firefox heeft een goede ondersteuning voor webstandaarden, iets dat van Microsoft's webbrowser niet gezegd kan worden. Microsoft heeft zijn browser de afgelopen jaren, op een popup-blocker na, amper verbeterd.

De browser van Microsoft staat zelfs bekend als een groot veiligheidsrisico. Veel virussen en spyware programma's komen via Internet Explorer op de PC. ActiveX is hier de boosdoener. Firefox kent geen ActiveX en heeft deze problemen niet.

Verder kan Firefox uitgebreid worden met extensies, geschreven door ontwikkelaars over de hele wereld. Zo heb ik zelf ook een paar extensies geschreven.

Doordat Firefox open source is, kunnen ontwikkelaars ook Firefox zelf verbeteren. En kan de browser geschikt gemaakt worden voor zowel Windows als de Macintosh en Linux.

Het marktaandeel van Microsoft's browser is hoger dan 90%. Velen zien Firefox ook als middel om Microsoft het monopoly op het Internet te onthouden.

Hoe dan ook, Firefox is een erg goede en vooral veilige webbrowser. En gratis.
