Java 1.5 is uit

do 30 sep 2004 20:31

Eindelijk is het dan zover: de nieuwe versie van Java is uitgebracht. Natuurlijk met nog meer mogelijkheden en nog sneller dan de vorige versie.

Voor mij als ontwikkelaar is dit natuurlijk mooi nieuws. Hoewel, ik moet nu wel weer mijn kennis bijspijkeren.

De kans is groot dat je geen flauw idee hebt wat Java is. Java is eigenlijk meer dan één ding. Het is een programmeertaal, waarin programmeurs programma's schrijven. Het heeft, simpel gezegd, een functiebibliotheek, met tienduizenden kant-en-klare stukjes programma die een programmeur direct kan gebruiken zonder het zelf te hoeven schrijven. En Java is een platform. Meestal schrijf je als programmeur óf voor Windows óf voor Macintosh óf voor UNIX. Het programma draait dan niet op een ander platform. Maar je kunt ook voor een denkbeeldige Java machine (Java Virtual Machine, JVM) schrijven. Omdat deze JVM beschikbaar is voor Windows, Macintosh, UNIX en nog wat platformen, kun je een Java programma op al deze platformen draaien. Je moet dan wel eerst de JVM downloaden en installeren. En van die JVM is dus vandaag een nieuwe versie uitgebracht.

Java op de desktop is nooit een echt succes geworden. Dat komt omdat de eerste versies vrij langzaam waren en men nu nog steeds (onterecht) denkt dat Java langzaam is. De juridische strubbelingen met Microsoft hebben ook niet meegeholpen. Op webservers en op wat duurdere mobiele telefoons is Java daarentegen erg populair.

Toch zijn er wel een aantal aardige Java programmaatjes die je op je eigen computer kunt gebruiken. Een aantal gratis voorbeelden vind je hieronder.
