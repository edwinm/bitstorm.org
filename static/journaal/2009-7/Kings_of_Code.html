Kings of Code

do 2 jul 2009 11:6

Maandag heb ik een presentatie gehouden over Advanced jQuery op de Kings of Code-conferentie, in een side-sessie georganiseerd door Fronteers. Ik mog de aftrap doen tegenover zo'n 100 bezoekers in het prachtige Tuschinski-theater.

De reacties waren wat wisselend. Ik kan me met mijn presentatietalent nog niet meten aan de meer geroutineerde sprekers op de conferentie. Daar tegenover heb ik veel positieve reacties gehad op de inhoud van de presentatie. Iedereen die ik sprak vond er wel iets interessants in.

Helaas heb ik maar één blogpost over de side-session gevonden, namelijk van de Zweed Daniel Nordstrom: Kings of Code 2009, day 1. Hij heeft ook wat Foto's op Flickr gezet.

De presentatie heb ik online gezet.
