VB .Net

ma 12 apr 2004 14:24

Als webontwikkelaar probeer ik een beetje bij te houden wat er allemaal speelt. Met UNIX, Java en XML lukt dat redelijk, maar Microsoft's nieuwe .Net is daarbij een beetje achtergebleven. Dus toen ik gratis een VB .Net resource kit kon bestellen, heb ik dat maar gedaan. Ik kon er ook nog een pet bij krijgen. Leuk voor L. ofzo.

Een paar weken later kreeg ik inderdaad een DVD boordevol documentatie, probeer-software en inderdaad: een pet. Een paar weken later ontving ik weer een DVD met een pet. En eergisteren onving ik wéér een DVD, maar nu met een T-shirt, want de petten waren op. Tja.
