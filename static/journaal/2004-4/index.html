VB .Net

ma 12 apr 2004 14:24

Als webontwikkelaar probeer ik een beetje bij te houden wat er allemaal speelt. Met UNIX, Java en XML lukt dat redelijk, maar Microsoft's nieuwe .Net is daarbij een beetje achtergebleven. Dus toen ik gratis een VB .Net resource kit kon bestellen, heb ik dat maar gedaan. Ik kon er ook nog een pet bij krijgen. Leuk voor L. ofzo.

Een paar weken later kreeg ik inderdaad een DVD boordevol documentatie, probeer-software en inderdaad: een pet. Een paar weken later ontving ik weer een DVD met een pet. En eergisteren onving ik wéér een DVD, maar nu met een T-shirt, want de petten waren op. Tja.

Spyware

zo 4 apr 2004 12:19

Ik vind dat mensen die dubbelklikken op onverwachte e-mail bijlagen of "Ja" aanklikken op onverwachte pop-ups voor straf een maand lang niet meer op het Internet mogen komen.

Met schaamte moet ik bekennen dat ik deze week ook slachtoffer ben geworden. Ineens was bij mij de "ISearch toolbar" in Internet Explorer geïnstalleerd. Daarnaast was ook de Google toolbar uitgeschakeld en het in- en uitschakelen van toolbars was onmogelijk gemaakt. Grrr.

Hoe was dit gekomen? In Firefox heb ik automatisch downloaden van bestanden aangezet. Als je op een bestand klikt, dan wordt deze zonder verdere vragen gedownload. Ik heb dit ook zo ingesteld dat deze bestanden op de desktop terechtkomen. Een paar dagen geleden zocht ik wat spelletjes voor L.. Die kwamen dus allemaal op de destop terecht. Maar blijkbaar ben ik ook op een pagina gekomen die er voor zorgde dat de ISearch toolbar werd gedownload. Door de Firefox instellingen kwam die dus automatisch, en zonder dat ik er erg in had, op mijn desktop.

Vervolgens ging ik de gedownloade spelletjes uitproberen. En daar zat dus de ISearch toolbar tussen. Zonder dat ik het doorhad was deze spyware geïnstalleerd. En had Internet Explorer een ISearch toolbar.

Na wat zoeken op Internet kwam ik erachter hoe je de toolbar kunt verwijderen. Iedereen met een ISearch toolbar raad ik aan onderstaande link te volgen.
