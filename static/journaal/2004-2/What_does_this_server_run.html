What does this server run?

zo 8 feb 2004 18:06

Soms wil ik graag weten welke software op een webserver draait. Bijvoorbeeld als een server overbelast is, wil ik weten of er toevallig Microsoft IIS op draait. Je kunt daar met allerlei UNIX en HTTP-commando's achter komen, maar het kan ook veel makkelijker.

Vanmiddag heb ik een klein PHP-scriptje gemaakt, die laat zien welke server er achter een bepaalde website hangt. Met een Bookmarklet kun je dan heel eenvoudig deze informatie oproepen.

Hoe gaat het in zijn werk? Klik met de rechter muisknop op de onderstaande link. Selecteer dan Toevoegen aan favorieten of Bookmark deze link. De Bookmarklet is dan geïnstalleerd. Mac gebruikers (zonder rechter muisknop) moeten de muisknop een paar seconden in gedrukt houden.

Als je vervolgens op een website komt waarvan je de server software wilt weten, dan selecteer je de bookmark/favoriet en het antwoord verschijnt in een pop-up venster.
