Jarx - the JAR executer

do 12 aug 2004 10:46

De afgelopen week heb ik weer een freeware programma geschreven. Ditmaal om Java programma's onder Windows gemakkelijk te kunnen opstarten. Veel Java programma's worden tegenwoordig verspreid in JAR-bestanden (met een .jar extensie). Zo een bestand dubbelklikken werkt op sommige computers, maar lang niet alle. Met Jarx kun je een JAR bestand altijd starten. En als je een oude Java-versie hebt, krijg je de mogelijkheid om de nieuwste versie te downloaden.
