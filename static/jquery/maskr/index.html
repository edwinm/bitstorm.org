Powerful templating with Maskr

  • Pure DOM-based templating
  • Using the power of jQuery
  • Supports JSON and XML
  • Very small: only 795 bytes minified and gzipped

Maskr is inspired by Pure.js, a DOM-based template engine. Maskr uses jQuery to make the engine more powerful and the file size smaller.

Maskr can use both JSON and XML as input data and uses a map to map the data to your DOM.
