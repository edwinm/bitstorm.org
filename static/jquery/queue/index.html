jQuery queue example

This page demonstrated the use of the jQuery queue function. It first fetches the feed url to show, then it uses the Google Feed API to fetch the RSS contents, then it does a call to the server to calculate the MD5 hash, which in turn is used to fetch the tags belonging the the article url from Delicious.

Take a look at the JavaScript source code to see how these functions are put together using the queue.

To learn more about the queue function, read my How to use the jQuery queue function weblog article.
