jQuery-plugins

Since 2007, I worked a lot with the jQuery JavaScript library. On this page you'll find jQuery plugins I made and are available as Open Source.

Color Animation

Use this plugin to finally be able to animate colors. Fade one color to another. Supported properties: foreground color, background color, border color and more.

Demo and download of Color Animation jQuery plugin

Shadow Animation

With this plugin, you can animate box shadows. Fade the color or change the position, blur or radius of the shadow or animate a combination of these properties.

Demo and download of Shadow Animation jQuery plugin
