Font Smoothing

Without font smoothing, large fonts like headings on a web page will look jagged or choppy. With Windows, you can easily make things better.

Open the Display Properties Control Panel

Font smoothing can be accessed from the Display Properties Control Panel. To open the Display Properties Control Panel, right-click on the desktop, then select Properties.
