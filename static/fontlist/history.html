Release history

FontList 1.6: October 27th, 2012
  1. Fixed bug in 1.5 in which IE complains it can't run JavaScript.
FontList 1.5: April 17, 2008
  1. Support for Vista.
  2. Double @-fonts not shown.
  3. Buttons shown on mouseover.
  4. Warning when JavaScript is not turned on.
  5. Warmer colors.
FontList 1.4: July 19, 2004
  1. Problem solved where fonts with a single or double quote in the name did not show.
  2. Show Postscript fonts (not only when Raster font is selected).
  3. Show number of fonts.
  4. Keyboard-shotcuts with Alt-key in FontList interface.
  5. Jump through the list by pressing a key.
  6. New icon.
FontList 1.3: Jan 24, 2002
  1. Now it's possible to change the style of a font to bold or italic and/or enlarge the font.
  2. The user can open a character map showing characters 32-255.
  3. Windows XP Theme support.
  4. Big icons (48x48) and alpha channel icons supplied.
FontList 1.2: Jan 5, 2002
  1. Now FontList also works when integrated desktop is not installed.
  2. Let user be able to choose install folder when installing.
  3. CSS added to prevent page-breaks in the sample text when printing (the browser must support CSS2).
  4. The sample text is now read from a separate file (Sample.txt), so you can change or replace the preset sample text.
  5. Sample fonts are a bit smaller.
FontList 1.1: Dec 4, 2001
  1. Corrected spelling.
FontList 1.0: Dec 3, 2001
  1. First release.
